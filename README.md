# jgarber623's dotfiles

## Usage

```sh
# Install GNU stow (https://www.gnu.org/software/stow/)
brew install stow

# Clone this repository
git clone https://github.com/jgarber623/dotfiles ~/.dotfiles

cd ~/.dotfiles

# Install symlinks for all packages using GNU stow
make install
```

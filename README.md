# jgarber623's dotfiles

This is how I work.

## Installation

First, launch Terminal.app and clone this repository:

```sh
git clone -b laptop/personal git@github.com:jgarber623/dotfiles ~/.dotfiles
```

Then, run the following commands:

```sh
cd ~/.dotfiles

bin/bootstrap
bin/install
```
